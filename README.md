# Dukaan AI Astro Site

Static marketing website for Dukaan AI built with Astro and Tailwind CSS.

Primary website domain: `dukaanai.co.in`
Android package: `com.dukaan.ai`
Business contact: `dukaanai.business@gmail.com`

## Stack

- Astro for static-first rendering and component architecture
- Tailwind CSS for fast, responsive styling
- Lucide Astro for lightweight icons
- Vite via Astro for local development and optimized builds
- Netlify or Vercel for CDN-backed deployment

## Local setup

1. Fix the local `npm` installation if needed.
2. Run `npm install`.
3. Run `npm run dev`.
4. Build with `npm run build`.

## Performance notes

- Most sections ship as HTML and CSS only.
- Feature images are lazy-loaded below the fold.
- Large marketing PNGs should be converted to WebP or AVIF before launch for better Lighthouse results.
